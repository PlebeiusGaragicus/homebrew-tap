# homebrew-tap

Homebrew formulas for [catmd](https://github.com/PlebeiusGaragicus/catmd).

Copy `Formula/catmd.rb` here when releasing — see [catmd AGENTS.md](https://github.com/PlebeiusGaragicus/catmd/blob/main/AGENTS.md).

## Install

```bash
brew tap PlebeiusGaragicus/tap
brew install catmd
```

## Update

```bash
brew update
brew upgrade catmd
```
